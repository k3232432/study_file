## NW LAN 과 WAN
LAN = LOCAL AREA NETWORK - 좁은지역(직접 구축) 
WAN = WIDE AREA NETWORK - 넓은 지역(임대|SK,KT,LG등)

WAN 을 받아준는 장치 = WAN ROUTER 
WAN ROUTER 와 CORE SWITCH 가 연결됨

즉 DISTRIBUTION ROUTER - WAN ROUTER - CORE SWITCH 를 통해서 연결이됨


WAN 설계 모델
- 허브 앤 스포크
- 풀메시
- 파샬 메시
